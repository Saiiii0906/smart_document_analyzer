# Smart Document Analyzer — Frontend

Built using:
- React
- AWS Amplify
- AWS API Gateway

## Commands

Install dependencies:
```
npm install
```

Start development server:
```
npm start
```

Build for production:
```
npm run build
```

## Folder Structure

- Upload documents
- Poll summary status
- Ask questions using RAG

Make sure to update:
`aws-exports.js` → with your Amplify + API Gateway config
